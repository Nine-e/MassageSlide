遇到的问题及解决方案

1.不同分辨率下页面的适配问题

解决方法:
在CSS中运用@media,讨论不同分辨率下的样式

						/*mobile*/
						@media screen and (max-width: 750px) {
						/*橫屏*/
						@media screen and (orientation: landscape) {
						.bet {
						width: 50%;
						}
						}
						/*竖屏*/
						@media screen and (orientation: portrait) {
						.bet {
						width: 80%;
						}
						}
						}
						/*ipad*/
						@media screen and (max-width: 1024px) and (min-width: 751px) {
						.bet {
						width: 50%;
						}
						}
						/*pc*/
						@media screen and (min-width: 1025px) {
						.bet {
						width: 20%;
						}
						}

2.字体

主要的字体有Montserrat和Lato
在CSS中通过 @font-face 引入 

					@font-face{
					    font-family: Montserrat;
					    src:url(../lib/Montserrat-Light.otf);
					}
					@font-face{
					    font-family: "Lato";
					    src:url(../lib/Lato/Lato-Regular.ttf);
					}
